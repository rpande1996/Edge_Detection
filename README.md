## Overview

This code demonstrates edge detection carried out by 2 methods; Gradient and Orientation

## Softwares

* Recommended IDE: PyCharm 2021.2

## Libraries

* Numpy 1.21.2
8 OpenCV 4.6.0
* Matplotlib 3.5.0

## Programming Languages

* Python 3.8.12
